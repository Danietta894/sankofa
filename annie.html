Annie Easley

Annie Easley foi uma cientista da computação, matemática e engenheira norte-americana que trabalhou na NASA e ajudou a desenvolver tecnologias fundamentais para programas espaciais e sistemas de propulsão. Sua contribuição foi essencial para o desenvolvimento de software de simulação para foguetes movidos a energia híbrida. Além disso, seu trabalho influenciou o desenvolvimento de tecnologias para a exploração espacial e para sistemas de energia renovável.

Principais Contribuições

  • Desenvolvimento de algoritmos para o programa de foguetes da NASA.
  • Pesquisa em sistemas de propulsão elétrica para missões espaciais.
  • Contribuições para o desenvolvimento de baterias utilizadas em veículos elétricos.
  • Atuação na inclusão de mulheres e pessoas negras na área da computação e engenharia.
  • Trabalho na análise de dados e desenvolvimento de software para a exploração espacial.
